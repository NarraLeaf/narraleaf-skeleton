# NarraLeaf-Skeleton

A utility for generating skeletons for NarraLeaf projects.

For NarraLeaf, see [NarraLeaf](https://github.com/NarraLeaf/NarraLeaf)

## Usage

```bash
npx narraleaf-skeleton my-skeleton-project
```
